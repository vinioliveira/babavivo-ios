#IOS-App 

Requirements 
---------------------------

* Xcode 
* Ruby
* Rubygems

Basic Setup
---------------------------

  Install the cocoapods

    $ gem install cocoapods

  Prepare project and install depedencies
    
    $ pod setup
    $ pod install

  After installation open the BabaVivo.xcworkspace and have fun.
  
  
  This is it for now.
